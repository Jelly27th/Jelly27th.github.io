#
趣事

#
九月十八日

翻了翻 B 站动态和微信朋友圈，才知道今天是九一八事变的纪念日，但历史我大多忘了。

#
被驯服的象

晚上妖风四起，冷得我鸡皮疙瘩都起来了。在五食堂田径场发呆时，也不知是否是我的错觉，总感觉附近的建筑物似乎被一层薄雾笼罩，这让我想起了一首歌的歌词。
“迷雾迷雾本迷雾 我惊觉自己在原地踏步”
“到底是谁把我心蒙住 不想再糊涂”
“迷路迷路迷了路 我就彻底被这团迷雾困住”
“谁能够指引我一条路带我走上正途”
我正在迷茫，渴望有指明灯

#
两只活宝

今天军训全程划水。我们第四数字方阵排完那几句话的彩排后，便一直坐着，闲到发霉。
“五五，收到请回答”
“五五信号已断连”
“四四，收到请回答”
“五四，四四己收到”
“……”
(5,5) 偶尔插一句话，(5,4),(4,4) 可真是一对活宝，(5,5) 感觉不无聊是这俩一直在聊，带动周围的氛围活跃。

#
调侃

早上集合时我们学院是最晚到的
“你不行啊”
女教官如是说
男教官好像误错了意，脸红了，抬不起头。
集合结束后，男教官竟没有狠狠地折磨我们
