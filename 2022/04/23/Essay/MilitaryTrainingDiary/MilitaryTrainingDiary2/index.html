#
大学军训的一天

#
5:30

跑步，跑步，还是跑步。

一大清晨起来就被要求早训，还有稀疏睡意，太阳老哥都没有我们起的这么早。而我和我的饥饿还在图书馆门前打着圈儿。

#
6:00

一个人头，两个人头，三个人头…… 毫无秩序。

早训完之后去买早餐，人群蜂拥而至，这是我最不愿遇到的情况，效率低。要是我会骑自行车，我更想去食堂买，可惜我不会，或许可以费些脚程，可我已疲倦了。

#
9:00

“脚跟着地，腿挺直，手臂伸直，肩关节放松”

“不要急！不要急！不要急！！！”

教官如是说。

我们正在专注的练齐步走，不得不说，小小的动作，放在团队里要求就高了。其一，一个人的动作标准可不行，还得所有人的动作都标准，其二，每个人的节奏都要一致，个子高的要慢一些，个子矮的要快一些，同时用余光观察队友动作。而我则总是快半拍或慢半拍。

#
14:00

中午一回宿舍倒头就睡。七月的炎热使我们下午不用训练，偶然看到发下来的学生手册，我想起来开学那场网上考试，现在想想挺无关紧要的，可我那时却很紧张，看了几小时的电子版学生手册，最后辞了暑期工回家了。

#
19:00

晚上仍然是训练，我还在傻傻的练正步，努力的向方正队靠齐，充满了羡慕，教官教了我们几首军歌，结果不一会儿已经有人被洗脑了。

#
22:00

我在群里阐述了我一天的军训生活。群成员里只有三人。
