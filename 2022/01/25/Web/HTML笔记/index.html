首页 Web

网站是网页的集合体
HTML 是超文本标记语言，所以只有标签，不是编程语言
浏览器内核是渲染引擎
web 标准构成：结构、行为、表现
标签关系：包含和并列
HTML 基本标签：HTML（head（title）body）
文档类型声明标签表示版本
编码类型、设备兼容性不容忽视
标签语义:
1、标题标签 h1-h6
2、段落标签 p
3、换行标签（强制换行）
4、文本格式化标签：推荐使用第一个
加粗 strong b
倾斜 em i
删除线 del s
下划线 ins u
5、无语义的标签（装内容）
div 占一行
span 跨距
6、图像标签和路径 img
img src="图像 URL"
图像属性:
src (必须)
alt
title
width
height
border
路径：
相对路径：图像下相对于 HTML 页面的位置，有同一级 上一级 下一级三种情况
绝对路径：完整的文件路径、网址路径
7、连接标签 a
语法格式 a href=“跳转目标” target=“目标窗口的弹出方式”> 文本或图像
target: _self (当前窗口打开页面) _blank (新建窗口打开页面)
href: 网址路径
连接分类:
外部链接
内部链接
空链接
下载链接
网页元素链接
锚点链接
8、注释标签！-- --
9、特殊字符
10、表格标签：展示数据
table 定义表格标签
tr 定义表格的行
td 单元格内的文字 td 定于表格的单元格
..........
tr
..........
table
11、表头单元格标签 th
表格属性（常用 CSS 实现）
对齐方式 align
有无边框 border
边框距离 callpadding
单元格距离 cellspacing
宽度 width
12、表格结构标签
头部 thead
主体 tbody

合并单元格：跨行合并 跨列合并
13、列表标签（布局）
无序列表 ul 只能包含 li
有序列表
自定义列表
表单标签（收集用户信息）
表单组成：表单域 表单控件 提示信息

新特性
语义化标签
头部标签 header
导航栏标签 nav
内容标签 article
定义文档的某个区域 section
侧边栏标签 aside
尾部标签 footer

多媒体标签
视频 video
音频 audio

input 标单
