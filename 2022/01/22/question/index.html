# Windows MingW64 安装报错 "the file has been downlodade incorrectly"

原因：我的网抽风或者官网抽风。
解决：直接下载源代码文件，并把 bin 文件目录添加至环境变量 path。

# clash 代理下 uwp 应用无法打开

uwp 应用举例：微软商店、Mircrosoft To Do。
原因:clash 流量处理方式对 uwp 的代理处理不一致，详情可看 UWP 应用联网。
解决：在 clash 的 General 中的 UWP Loopback 选择你要放行的 uwp 应用即可。
